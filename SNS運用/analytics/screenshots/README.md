## 📌 用途

X投稿の **初動（5分後 / 60分後 / 24時間後など）** のimp / RT / 保存数を画像で残す場所。

- 後でnote記事や振り返りで「この投稿の初動こうでした」と見せたい時に使う
- 数値だけでよい時は週次戦略メモ（`WNN戦略メモ.md`）に直接テキストで追記し、ここには保存しなくてよい

## 🏷 命名規則

```
WNN-{投稿種別}-{投下時刻 or 連番}-{経過時間}.png
```

例:
- `W18-series02-1230-60min.png`  → W18 連投シリーズ② 12:30 投下の60分後
- `W18-routineA-0900-24h.png`     → W18 ルーチンA 9:00 投下の24時間後
- `W18-thread-news-1900-5min.png` → W18 ニューススレッド 19:00 投下の5分後

## 📥 ファイル参照の書き方

action-YYYYMMDD.md などから参照する時はObsidian埋め込みで:

```markdown
![[SNS運用/analytics/screenshots/W18-series02-1230-60min.png]]
```

## 🧹 整理ポリシー

- **保管期間**: 該当週の振り返りが終わったら任意で `_trash` へ移動可
- **永久保存**: note記事に使った画像は `_ kiwami/note-assets/` などへコピー（ここはあくまで一時保管）
