## 1. そもそもNemoClawって何者？

NemoClawは、NVIDIAがGTC 2026（サンノゼで3月16〜19日開催）で発表した**AIエージェントを作るための全部乗せセット（スタック）**です！
OpenClawの「ずっと賢く動き続けるAIエージェント（Claw）」を、企業が安心して使えるようにセキュリティとプライバシー保護を強化したものですね。

ポイントは、ただのアプリじゃなくて「スタック（セット）」なところ！NVIDIAの**OpenShellランタイム**と賢いAIモデル**Nemotron**がセットになってて、コマンド一つでサクッと導入できるようになってるみたいです！
今はまだ**アルファ版（初期テスト段階）**だけど、ライセンスは商用利用もしやすい**Apache 2.0**で公開されてます！

## 2. これまでのあらすじ（OpenClawの爆誕）

NemoClawのスゴさを知るには、親玉の「OpenClaw」のヤバい歴史を知るのが一番です！

- **2025年11月**: オーストリアの天才エンジニア **Peter Steinbergerさん** が、WhatsApp経由で仕事をお願いできるAI「Clawdbot」を**たった1時間**で作って公開しちゃいました💦
- **2026年1月27日**: Anthropicから「名前が似てる！」って怒られて「Moltbot」に改名（笑）。
- **2026年1月30日**: そこからさらに「**OpenClaw**」に改名して、オープンソースだったこともあって口コミで爆発的に広まりました！
- **2026年2月14日**: なんと開発者のSteinbergerさんがOpenAIに引き抜かれちゃって、プロジェクトはオープンソース財団に引き継がれました。
- NVIDIAのジェンセン・ファンCEOも、このOpenClawを「おそらく史上最も重要なソフトウェアリリース」って大絶賛してます！

OpenClawは「自分のパソコンで動くAIエージェント」を一気に広めた神ツールだけど、同時に「勝手に色んなデータ触っちゃうかも…」っていうセキュリティの不安があったんですよね。それをNVIDIAが全力で解決しに来たのが「NemoClaw」ってわけです！

## 3. すごい仕組み（4層アーキテクチャ）

NemoClawは、大きく4つのパーツで安全を確保してます！

### 3-1. Plugin（プラグイン係）
- TypeScriptで作られた操作ツール！
- AIエージェントの起動とか接続、今の状態確認、記録（ログ）を残す仕事をやってくれます。

### 3-2. Blueprint（設計図係）
- 管理されたPythonのプログラム群ですね。
- AIを隔離する「箱（サンドボックス）」を作ったり、ルールの適用、AIモデルの設定なんかを取り仕切ってくれます。

### 3-3. Sandbox（安全な遊び場）
- OpenShellっていうコンテナの中で、OpenClawを完全に隔離して動かします！
- ルールに従って、ネットワークを使わせるか、どのファイルにアクセスさせるかを厳しく監視してくれます。

### 3-4. Inference（賢い頭脳への通信手段）
- NVIDIAのクラウドにある強力なAIモデルへの通信を、ゲートウェイを通すことで安全かつスムーズに繋いでくれます！

ソースコードの中身は TypeScript 34.9% / JavaScript 30.2% / Shell 28.1% / Python 5.5% っていう割合だそうです。

## 4. 鉄壁の防衛網（セキュリティアーキテクチャ）

### セキュリティレイヤー一覧

#### 🌐 Network（ネットワーク）
- **何をしてるか：** 許可してない外部への怪しい通信を全部ブロック！
- **ルールの変更：** 途中で変更OK

---

#### 📁 Filesystem（ファイルシステム）
- **何をしてるか：** 隔離部屋（`/sandbox`と`/tmp`）以外は絶対見せない！
- **ルールの変更：** ガチガチに固定

---

#### ⚙️ Process（プロセス）
- **何をしてるか：** 勝手に強い権限を持ったり、危険な命令を出すのを防ぐ！
- **ルールの変更：** ガチガチに固定

---

#### 🤖 Inference（推論）
- **何をしてるか：** AIモデルへの質問ルートを安全な道だけに絞る！
- **ルールの変更：** 途中で変更OK

ちゃんと宣言したルールに基づいて管理されてて、もし登録されてない場所に通信しようとしたら「人間が許可ボタンを押すまで待て！」っていうワークフローが働くみたい。めっちゃ安心！

## 5. キモはこれ！「プライバシールーター」

NemoClawが他と違う決定的な理由が、この**「プライバシールーター」**です！

- NVIDIAが買った「Gretel」っていう強力なデータ保護企業の技術をそのまま使ってます！
- ChatGPTとか外部の強力なAIモデルに質問する前に、**プロンプトの中にある個人情報（名前とか電話番号）をAIが自動で隠してくれる**んです！凄くないですか💦
- これのおかげで、社内の安全なAIだけでなく、外の賢いAIも安心して仕事に使えるようになります！

**ハイブリッドな使い方**:
- 普段の仕事: ローカルにあるNVIDIAのAI（Nemotronモデル）が安全・爆速で処理。
- めっちゃ難しい仕事: このルーター経由で外の超賢いAIにお願いする。
この使い分けができるから、プライバシーは守りつつ、AIがどんどん新しいことを覚えて実務をしてくれるんです！

## 6. 対応ハードと「Mac非対応」の悲劇…😢

NemoClawは「ハードウェア非依存」なんで、NVIDIAだけじゃなくAMDやIntelのプロセッサでも動くみたいです！すごっ！

NVIDIA機なら以下を公式サポート：
- **個人用**: GeForce RTXが載ってるパソコン
- **プロ用**: RTX PROが載ってるワークステーション
- **企業用最強**: DGX Stationとかのスーパーコンピュータ

**…で、Macは！？**
気になってシステム要件（Ubuntu 22.04 LTS必須とか、NVIDIA OpenShell必須とか）をめっちゃ調べたんですが…Apple SiliconにもmacOSにも記述が一切ないんです…💦
つまり現在のアルファ版では、**Mac（macOS / Apple Silicon環境）でのネイティブ利用は実質不可能**っぽいです！！
Macユーザーの私としては、世界最高のAIツールが目の前にあるのに触れないなんて、悲しすぎてめちゃくちゃ泣いてます😭 Mac対応版、首を長ーくして待ってますNVIDIAさん…！

## 7. どうやってインストールするの？（※Mac以外）

**動かすための条件**:
- Ubuntu 22.04 LTS以降のOS（←ここでもうMacは脱落…😢）
- Node.jsのバージョン20以上（22がおすすめ）
- Docker（準備しといてね）
- NVIDIA OpenShell（事前に入れておく）

**サクッとインストール（Linux環境の人向け）**:
```bash
curl -fsSL https://nvidia.com/nemoclaw.sh | bash
```
これを入れると便利なウィザードが立ち上がって、対話しながらセットアップできるみたいです！

**よく使うコマンド集**:
- `nemoclaw onboard` — セットアップ開始！
- `nemoclaw <name> connect` — 隔離部屋へのアクセス！
- `nemoclaw start/stop/status` — 動かす・止める・状態確認！
- `openshell term` — 操作画面を立ち上げる！
