## 1. Googleが放った刺客「Gemma 4」、Apache 2.0で完全商用利用を解放！

**概要**
Googleが最新のオープンAIモデル「Gemma 4」をリリースしました。ライセンスがApache 2.0に変わり、完全な商用利用が可能に。4つのサイズ展開（2Bから31B）で、一番小さいモデルはスマホやRaspberry Piでも遅延なく動きます。

**実務観点での感想**
昨日の記事でも触れたんですけど、改めて言わせてください。**これ、本当にデカい**💡

Apache 2.0って、オープンソースの世界では最強のライセンスなんですよ。企業が法的リスクなく自社サービスに組み込める。法務部門が「ちょっと待って」ってならない。これだけで、AI内製化を検討してた企業のハードルが一気に下がるわけです。

しかも驚くべきは**パラメータ効率**。ベンチマークで、自身の20倍以上のサイズを持つ旧来モデルを圧倒してるんですよ。2Bモデルがスマホでオフライン動作するってことは、「Edge AI」がいよいよ現実のものになってきたってこと。

**追加情報**
動画・画像・音声をネイティブ処理できるマルチモーダル対応も入ってます。テキストだけじゃなくて、視覚も聴覚も処理できる小型モデルが、スマホの中で完結して動く時代。先日のTurboQuantの話と合わせると、Googleは「モデルの効率化」と「モデルの解放」の両面で攻めてきてて、エコシステムごと取りに来てる印象です💦

🔗 **情報源：** The Register / AMD Developer

## 2. Microsoft、日本に100億ドル（約1.5兆円）のAI巨額投資！アジアのAI拠点化が加速

**概要**
Microsoftが日本国内のAI・クラウドインフラ拡充に100億ドル（約1.5兆円）を2029年までに投じることを発表。ソフトバンクやさくらインターネットと提携し、データセンター増設と100万人以上のAI人材育成を進めます。

**実務観点での感想**
昨日も取り上げたんですけど、このニュースは何回でも言いたい。日本でAI関連の仕事をしてる人にとっては**めちゃくちゃ大きな転換点**ですよ💡

だって今まで「日本はAI後進国」って言われてきたじゃないですか。計算リソースが海外と比べて圧倒的に足りなくて、データセンターも少なくて。それが1.5兆円のインフラ投資で一気に底上げされるわけです。

ポイントは「データの主権を保ったまま」ってところ。国内にデータセンターが増えるってことは、機密データを海外に出さずにAI処理ができるようになる。金融機関とか官公庁みたいなセキュリティ要件が厳しい組織にとっては、AI導入のハードルが劇的に下がるんですよね。

**追加情報**
さくらインターネットとの提携も注目。国内クラウドベンダーがMicrosoftと組むことで、大企業だけじゃなくて中小企業やスタートアップにもGPUリソースが行き渡る仕組みが作られる。昨日の記事で書いた「推論コストで勝負が決まる」って話とリンクしてて、日本企業もその「勝負のフライホイール」に乗れる可能性が出てきました！
