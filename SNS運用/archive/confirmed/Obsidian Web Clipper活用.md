X開始53日でフォロワー6000人になった記念に、僕が実践してる情報収集のやり方を紹介する。

1. Obsidian Web Clipperで情報を保存
2. Obsidianに知識が蓄積される
3. Claudianを導入し「これでポスト作って」と話す
4. 投稿案が一瞬で出てくる

拡張機能「Obsidian Web Clipper」と検索。 
これをChromeやSafariに入れると、気になったYouTubeやXの投稿を開いたまま1クリックするだけで保存できる。 
保存するたびに知識が蓄積されていき、自分の脳みそを拡張できる。

気になった情報をObsidianに溜め続けると、自分だけの情報源からClaudeがポストを作ってくれるから、誰とも被らない唯一無二の発信ができる。 
ネタ切れで悩むこともなくなるし、情報の質が一段上がる→信頼される=収益が出やすくなる。
