## 投稿①（朝）

> 📸 画像を1枚追加

> 🖼️ **サムネイル文言**
> - **フック文（大）**: 4月「できなかったこと」を3つ書く
> - **サブ文（小）**: 5月の道筋は30分で見える

```
5月1日の朝に、4月「できなかったこと」を3つだけ書いた。

①週次の振り返りを後回しにした
②note記事の保存数が想定の半分以下
③Threadsの早朝枠を全部逃した

これだけで、5月の優先順位が勝手に決まった。

「やったこと」を整理するより、「できなかったこと」を3つ書く方が、次の月の道筋が早く見える。

あなたが4月「できなかったこと」、ひとつだけ教えて👇
```

> **メモ:** 月初の朝に最適な振り返り型。「やった/やってない」論じゃなく、「できなかったこと」起点で書く方が次に活きる。

## 投稿②（夜）

```
金曜の夜、週の振り返りで一番効くのは「数字」じゃなくて「驚き3つ」。

①想定外に良かったこと1つ
②想定外にダメだったこと1つ
③そのどちらでもない、引っかかったこと1つ

数字は冷静になれるけど、驚きは記憶に残る。

来週も同じパターンが来た時、「あ、これ先週もあったな」と気づける。
```

> **メモ:** 週次振り返りの王道は「数字」だけど、それより「驚き」の方が次に活きるという仮説。Day Oneプロモ並走中なので売り込みは入れない。
