### Prompt 01：取引先への謝罪メール

> 状況を入力するだけで、誠意の伝わる謝罪メールが即完成。

```
あなたはビジネスメールの専門家です。
以下の情報を元に、取引先への謝罪メールを作成してください。

・相手：【取引先の会社名・担当者名】
・状況：【例：納品が3日遅延した】
・原因：【例：社内の確認漏れ】
・対策：【例：今後はダブルチェック体制を導入】

【条件】
- 件名も提案すること
- 誠意を示しつつ、具体的な改善策を必ず明記
- 200〜300文字で簡潔にまとめること
```

### Prompt 02：上司への報告・相談メール

> 悪い報告もスムーズに。結論ファーストで上司を安心させる。

```
あなたはビジネスコミュニケーションの専門家です。
以下の情報を元に、上司への報告・相談メールを作成してください。

・報告内容：【例：A社プロジェクトの進捗が1週間遅れている】
・原因：【例：先方の仕様変更があったため】
・現状の対策：【例：チーム内で優先度を上げて対応中】
・相談事項：【例：追加リソースの投入可否を判断いただきたい】

【条件】
- 結論（現状）→ 原因 → 対策 → 相談の順で構成
- 感情的にならず、事実ベースで記述
- 200文字以内で簡潔に
```
