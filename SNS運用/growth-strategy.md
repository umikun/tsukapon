## フェーズ別戦略

### Phase 1: 冷起動期（現在〜1,000フォロワー）

**核心タスク**: 発信習慣の確立 + 最初の100人の「真のファン」を獲得

#### 毎日のルーティン（1日60分）

| 時間 | アクション | 所要時間 |
|------|----------|---------|
| 朝 9:00 | 投稿①を公開（平日9AM-2PMがベスト） | 5分 |
| 朝 9:15 | 大手AI系アカウント5-10件のリプ欄に高品質リプ | 20分 |
| 昼 12:00 | リプへの返信（対話=150倍の重み） | 10分 |
| 夜 20:00 | 投稿②を公開 | 5分 |
| 夜 20:15 | リプ欄への追加リプ + DM対応 | 20分 |

#### 重点施策

**1. 人気アカウントのリプ欄でリーチを借りる（最重要）**
- AI系の大手アカウント10-20人をリスト化し通知ON
- 投稿後15分以内に200-400字の「ミニnewsletter式リプ」を残す
- 1つの高品質リプ = 数千回のインプレッション獲得可能

**2. Premium加入（必須投資）**
- TweepCred（アカウント信頼スコア）が即座に+100
- 非Premiumの外部リンク投稿のインプレッション中央値 = ほぼゼロ
- 4倍のフォロワーFeedブースト + 2倍の非フォロワーFeedブースト

**3. プロフィール最適化**
- Bio: 「誰に」「何を」「なぜ自分が」を明確に
- 固定ツイート: 最もエンゲージメントの高い投稿を週1で差し替えテスト
- プロフ→noteへの導線を明確に

**4. コンテンツ配分**
- 75%: 広く刺さるコンテンツ（誰でも共感できるAI時短Tips）→ 新規獲得
- 25%: 深掘りコンテンツ（自分の経験・失敗談・数字公開）→ ファン定着

#### KPI目標
| 指標 | 目標値 |
|------|--------|
| 日次投稿数 | 2-3件 |
| エンゲージメント率 | >2% |
| 週次フォロワー増加 | +35-70人 |
| 他アカウントへのリプ | 1日10件以上 |

### Phase 2: コンテンツ検証期（1,000〜10,000フォロワー）

**核心タスク**: 高パフォーマンスコンテンツの型を見つける + テンプレート化

#### 追加施策

**1. 週1回のThread投稿（8-12ツイート）**
- Threadは通常投稿より47%高いエンゲージメント
- 構成: Hook → Main Points → TL;DR → CTA
- Cole式7テンプレートを活用:
  - Framework / Story / Actionable / Curation / Lessons / Mistakes / Contrarian

**2. コンテンツ複製フライホイールの本格稼働**
```
X短文投稿（検証）
  ↓ データが良ければ
Xスレッド（展開）
  ↓ データが良ければ
note長文記事（深掘り）
  ├── Threads用に短縮版
  ├── Instagramリール
  └── 次の投稿ネタに還元
```

**3. newsletter導線の確立**
- 全Threadの最終ツイートにnewsletter登録CTA
- 「Xは発見エンジン、newsletterは深い関係」
- メールリストは「自分が所有する資産」（アルゴリズム変更に左右されない）

**4. データ駆動の改善**
- 週1回: トップ3投稿を分析し共通パターンを抽出
- 月1回: コンテンツ種類別のROIを分析し、低パフォーマンスを切り捨て
- 高パフォーマンス投稿の構造をテンプレート化

#### KPI目標
| 指標 | 目標値 |
|------|--------|
| 週次Thread | 1-2本 |
| エンゲージメント率 | >3% |
| noteへの誘導 | 週10件以上のクリック |
| newsletter登録 | 週5-10件 |

## Xアルゴリズム活用チェックリスト

### 毎投稿前に確認

- [ ] Hookは2行以内で注意を掴んでいるか？
- [ ] 外部リンクは本文に含まれていないか？（リプ欄へ）
- [ ] ハッシュタグは2個以下か？
- [ ] Reply/Bookmarkを促すCTAがあるか？
- [ ] 投稿時間は平日9:00-14:00か？（火・水がベスト）

### 投稿後15分以内

- [ ] 最初のリプに反応したか？（対話=150倍の重み）
- [ ] 外部リンクが必要なら第1リプに貼ったか？

### 週次チェック

- [ ] 大手アカウントのリプ欄に10件以上リプしたか？
- [ ] Threadを1本以上投稿したか？
- [ ] 固定ツイートの差し替えテストをしたか？
- [ ] トップ3投稿の共通パターンを抽出したか？

## クロスプラットフォーム戦略

### X（メイン戦場）
- 役割: 新規発見 + エンゲージメント構築
- 頻度: 毎日2-3投稿 + 週1Thread
- 特徴: アルゴリズム重視、Reply/対話が最重要

### Threads（サブ戦場）
- 役割: Xコンテンツの二次展開 + 別チャネルでのリーチ
- 頻度: 毎日1-2投稿（X投稿のリライト）
- 特徴: よりカジュアル、Instagram経由のフォロワー流入

### note（深掘り + 収益化）
- 役割: 長文コンテンツ + 有料記事 + メンバーシップ
- 頻度: 週2-3記事
- 特徴: XからのCTAで誘導、SEOでの流入も狙う

### Instagram（ビジュアル展開）
- 役割: リール動画でのリーチ拡大
- 頻度: 週3-5リール
- 特徴: day1-10の動画テンプレートを活用して量産

## 避けるべき反パターン

| やってはいけないこと | 理由 |
|-------------------|------|
| 互フォロー/互いいねグループ | TweepCredが暴落、アルゴリズム降格 |
| 全ての話題に反応 | ポジショニングがぼやける。自分のテーマに関連する話題だけ |
| 外部リンクを本文に直貼り | リーチ-30〜50%。リプ欄へ |
| ハッシュタグ3個以上 | リーチ-40%、スパム判定のリスク |
| リプに返信しない | 対話=150倍の重み。最大のアルゴリズムレバーを放棄 |
| 長すぎるThread（15+ツイート） | 離脱率急上昇。8-12が最適 |
| Hookに時間をかけない | 最初の15分で生死が決まる。創作時間の50%をHookに |

## 差別化ポジショニング

### 「AI時短の人」としてのブランド確立

**Interest Stack（興味の掛け合わせ）**:
- AI活用 × 時短 × 副業 × リアルな数字公開

**Snow Leopard戦略**:
- 「AI活用」だけでは競合多数
- 「40代会社員がAIで定時退社→副業月5万」という具体的ストーリーが差別化

**4Aネタ選びマトリックスの活用**:
