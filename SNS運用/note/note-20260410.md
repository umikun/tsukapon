## 1. Google「Gemma 4」発表！Apache 2.0で全機能を完全開放──"オープンソースAI元年"がついに来た

> **この章のポイント**
> - Apache 2.0ライセンスで商用利用が完全自由に
> - スマホでオフライン動作する「2Bモデル」が登場
> - 法務リスクなく企業がオンプレで自律エージェントを構築可能

### 概要
Googleが最新のオープンモデル「Gemma 4」をリリース。最大の特徴は、ライセンスを**Apache 2.0**へ移行し、完全な商用利用を認めた点です。4つのサイズ展開（2B〜31B）があり、最小モデルはスマホでも遅延なく動作。ベンチマークでは自身の20倍以上のパラメータを持つ他社モデルを圧倒する「知能密度」を記録しました。特定のハードウェアに依存せず、オンプレミスでの自律エージェント構築を強力に支援します。

### 実務観点での感想
これね、冷静に考えるとヤバいんですよ。

Apache 2.0ライセンスってことは、**企業の法務部門が「これ使ってOK」って言える**ってこと。今まで「オープンモデル使いたいけどライセンス条件が複雑で…」って渋い顔されてた壁が、一夜にして消えました。

しかも31Bモデルがオンプレで動くってことは、**機密データを外に出さずに高度な推論ができる**ようになったってこと。金融、医療、自治体──「クラウドに上げたくない情報」を抱えてる業界にとって、これは本当に革命です。

スマホで2Bモデルがオフライン動作するのも、地味にデカい。**通信が切れる現場（工場・建設・災害時）でAIが使える**ってことだから、活躍の場が一気に広がります。

### Gemma 4のポイントまとめ
- Apache 2.0ライセンスで商用利用が完全自由
- 2B〜31Bの4サイズ展開、スマホでもオフライン動作
- オンプレミスで「データを外に出さない」自律エージェント構築が可能
- 旧モデルの20倍以上のパラメータを持つ他社製品を性能で上回る

🔗 **情報源：** Google Blog / AMD Developer
