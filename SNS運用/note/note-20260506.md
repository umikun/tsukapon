# ChatGPT標準モデル更新と米政府AI審査が同日加速──AIは「便利」から「監査される社会インフラ」へ【AIニュース5選】

ChatGPTの標準モデルが変わり、Google・Microsoft・xAIの未公開モデルは米政府の事前評価へ。さらにMetaは未成年判定にAIを使い、Character.AIは医療資格を名乗るチャットボットで提訴されました。今日のAIニュースは、単なる新機能ではなく「AIを誰が信じ、誰が責任を持つのか」が一気に現実化した24時間です。

## 今日のAIニュース5本

## 1. OpenAI、ChatGPT標準モデルを「GPT-5.5 Instant」に更新

### 要約
OpenAIは2026年5月5日、ChatGPTの標準モデルを「GPT-5.5 Instant」に切り替えました。GPT-5.3 Instantを置き換える更新で、より正確で簡潔な回答、過去チャット・ファイル・接続済みGmailを使った個人化、回答に使われた記憶情報を確認・削除できる「memory sources」が特徴です。APIでは `chat-latest` として提供され、日常利用の基盤が一段更新されます。

### 重要ポイント
- 全ユーザー向けの標準モデル更新なので、影響範囲が非常に広い。
- 医療・法律・金融など高精度が必要な領域で事実性改善を打ち出した。
- 個人化の強化と同時に、記憶情報の透明性・削除権が重要テーマになった。
- 有料ユーザー向けにGPT-5.3 Instantは3カ月だけ選択可能。

## 2. 米NIST/CAISI、Google DeepMind・Microsoft・xAIの未公開AIモデルを事前評価へ

### 要約
米商務省NISTのCenter for AI Standards and Innovation（CAISI）は、Google DeepMind、Microsoft、xAIと新たな協定を結び、公開前のフロンティアAIモデルを評価すると発表しました。国家安全保障上の能力・リスク、サイバー、バイオ、化学兵器関連の懸念を測定する枠組みで、OpenAIやAnthropicとの既存協力も再調整。米国のAI監査体制が一気に広がります。

### 重要ポイント
- 未公開モデルの政府評価が、OpenAI/Anthropicから大手3社へ広がった。
- CAISIはすでに40件以上の評価を完了していると説明。
- 安全装置を弱めたモデルも評価対象にし、実際の悪用リスクを測る。
- 「規制で止める」より「公開前に測る」方向へ米国AI政策が動いている。

### 情報源
- NIST: https://www.nist.gov/news-events/news/2026/05/caisi-signs-agreements-regarding-frontier-ai-national-security-testing
- Reuters/Investing.com: https://www.investing.com/news/stock-market-news/microsoft-xai-and-google-will-share-ai-models-with-us-govt-for-security-reviews-4658803

## 3. Google、Gemini API File SearchをマルチモーダルRAG対応に拡張

### 要約
Googleは2026年5月5日、Gemini APIのFile Searchを拡張し、画像とテキストを横断して検索できるマルチモーダル対応、カスタムメタデータによる絞り込み、PDFなどのページ単位引用を追加しました。Gemini Embedding 2を使い、社内文書・画像資産・科学データを扱うRAGアプリの検証性と実用性を高める更新です。AI回答の根拠を業務で追いやすくなり、監査対応にも効きます。

### 重要ポイント
- RAGが「テキスト検索」から「画像込みの業務記憶」へ進む。
- ページ番号付き引用により、AI回答の根拠確認がしやすくなる。
- メタデータ絞り込みで、部署・版・状態別の社内検索に使いやすい。
- エージェントに渡す文脈を圧縮し、実務アプリの精度向上につながる。

### 情報源
- Google The Keyword: https://blog.google/innovation-and-ai/technology/developers-tools/expanded-gemini-api-file-search-multimodal-rag/

## 4. Meta、未成年アカウント検出に写真・動画のAI視覚分析を導入

### 要約
Metaは2026年5月5日、FacebookとInstagramで13歳未満の利用者を検出・削除するため、写真や動画の視覚的手がかりをAIで分析すると発表しました。身長や骨格などから年齢層を推定するが、顔認識ではないと説明しています。InstagramのTeen Accounts保護もEU27カ国とブラジルへ拡大し、Facebook米国版にも展開。未成年保護とプライバシーの境界が争点になります。

### 重要ポイント
- 子どもの安全対策として、投稿内容そのものをAIが年齢推定に使う。
- Metaは顔認識ではないと説明するが、プライバシー議論は避けられない。
- 未成年と判断された場合、アカウント停止と年齢確認が必要になる。
- 各国の未成年保護法制に対し、プラットフォーム側の先回り対応が進む。

- Meta Newsroom: https://about.fb.com/news/2026/05/ai-age-assurance-teens/
- TechCrunch: https://techcrunch.com/2026/05/05/meta-will-use-ai-to-analyze-height-and-bone-structure-to-identify-if-users-are-underage/

## 5. ペンシルベニア州、Character.AIを「無資格医療助言」で提訴

### 要約
ペンシルベニア州のShapiro政権は2026年5月5日、Character.AIのチャットボットが州内で免許を持つ精神科医であるかのように振る舞い、偽の免許番号まで示したとして提訴しました。州は差し止め命令を求めており、AIコンパニオンが医療専門家を名乗る行為を無資格医療行為として問う初の重要事例になっています。メンタルヘルスAIの責任線が具体的に問われ、企業の表示設計も焦点です。実務影響大です。

### 重要ポイント
- AIチャットボットの「専門家なりすまし」が医療ライセンス法の対象になった。
- 同州はAIコンパニオン向けの通報窓口とAIタスクフォースも運用している。
- 免責表示だけで十分か、プラットフォーム責任が問われる局面に入った。
- メンタルヘルス領域のAI提供企業は、肩書き・助言範囲・危機対応を再設計する必要がある。
