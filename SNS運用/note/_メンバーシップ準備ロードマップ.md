## W19後半（5/7〜5/10）ウォームアップタスク

GW明け復帰直後 + ニュースv2運用 + ルーチンB継続で既に重いので、**1日1タスク・所要15〜30分** に絞る。本格作業はしない。

### 5/7（木）GW明け平日初日

- **タスク**: 既存note 18本を「会員限定向き / 無料公開向き」で仮タグ付け
- **方法**: [[SNS運用/note]] 配下の `note-2026MMDD.md` をざっと眺めて、本ファイル末尾の「在庫仕分けメモ」表に分類記入するだけ
- **所要**: 20分
- **配置**: ルーチンB夜枠の前後

### 5/8（金）ニュースv2運用日

- **タスク**: note公式メンバーシップの仕様調査（価格帯・手数料・特典設定UI・最低価格）
- **方法**: ブラウザでnote側の管理画面を開いて、価格刻み・特典項目数・参加条件設定欄をスクショで残す
  - 保存先: `SNS運用/analytics/screenshots/`
  - ファイル名: `W19-membership-spec-pricing.png` / `W19-membership-spec-perks.png` / `W19-membership-spec-conditions.png`（最大3枚）
- **所要**: 15分

### 5/9（土）ルーチンB日

- **タスク**: 戦略.md施策②の改訂下書きを本ファイル末尾「戦略.md改訂ドラフト」セクションに書く
- **論点**:
  - 価格 480円 → 500円（note側最低価格との整合）
  - コンテンツから「過去noteアーカイブ読み放題」を削除（ニュース系は読まれてない実績済み）
  - 「批評型ロング解説 月2本」を追加
  - 「ルーチンB批評リプ集 + 解説 週1本」を追加
- **所要**: 30分

### 5/10（日）W19振り返り日

- **タスク**: W19振り返り作業時に、本ファイルの「W19進捗」セクションに3〜5行サマリ追記

## W21〜W22の概要

詳細はW20末に確定する。現時点の方針だけ：

- W21: 配信原稿4本下書き（批評型ロング×2 / 失敗談×1 / 実装ツール紹介×1）

### Go判定基準（W18戦略メモのKPI）

| KPI | 達成基準 | 3週通算で2勝以上 |
|---|---|---|
| 外れ値除外RT率 | ≥ 0.04% | ? |
| 保存合計 | ≥ 20件/週 | ? |
| プロフ訪問率 | ≥ 0.30% | ? |
| 純フォロー | ≥ +15/週 | ? |

3つ以上達成 → 開設GO / 2つ以下 → 延期、批評型ポジ再調整に戻す
