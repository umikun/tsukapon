## A. プロフィール文（書き換え3案）

### 案A1：「ツッコミ役」直球型

```
AI界隈の煽りに毎朝ツッコむ人 / 元・月80h残業マン → 定時退社へ
"神プロンプト100選"より"原理1個"派
毎朝AIニュースをnoteで深掘り / 週末は調査ノート
👇 ピン留めから読むと早いです
```

→ メリット: ポジショニングが一発で伝わる。「中の人」に飽きた層を拾える
→ デメリット: 「批評ばっかりの人」と思われると保存系コンテンツが弱まる

### 案A2：「ツッコミ + 実務」両立型 ★推奨

```
AIニュースの"煽り"を毎朝ツッコみつつ、自分の仕事に効く部分だけ抜き出す人
元・月80h残業 → 定時退社（プロンプト30個でやった話はピン留めに）
毎朝note更新 / 週末は調査ノート
"なぜ効くか"が分からない神プロンプトは保存しない派 😎
```

→ メリット: 批評ポジ ＋ 実務実績の両方を1画面に。保存・フォロー両方の動機を作れる
→ デメリット: 文字数ギリ（要X側で計測）

### 案A3：「読者の自分ごと化」型

```
"AIで人生変わった"系の話を9割疑い、1割だけ拾って自分の仕事に使う人
元・月80h残業 → 定時退社（その1割の話はピン留め）
毎朝AIニュースnote / "なぜ効くか"派
😭🤔🫠 で煽りに反応してます
```

→ メリット: 読者目線がいちばん強い。「自分と同じ温度」と思われやすい
→ デメリット: ややネガ寄りに見える可能性

### 案B3：「煽り検証シリーズ」のハブ投稿固定型

W18で連投する4本の批評シリーズを束ねた「目次ツイート」をピン留め。

