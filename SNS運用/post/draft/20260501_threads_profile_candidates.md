> 起点: X側プロフィールが 5/4(月) W19初日に「AI界隈の煽りに毎朝ツッコむ人」型へ批評型ポジションへ書き換わるため、**Threads側も人格整合をとる**かどうかを判断するための比較案。X単独改装だとXとThreadsで人格が分裂し、フォロー導線が混乱するリスクがあるため、3案を作って選定する。

---

## 起点となる事実

- **X側の改装方向**: 4/26リプ（@09pauai宛）の単独大爆発（imp 14,430 / RT 18）を再現するため、**「煽りに冷静にツッコむ人」**ポジションへ寄せる
- **Threads側の特性**:
  - 検索流入が弱く、**フォロワー間のコミュニティ感**でリーチが伸びる
  - テキスト主体・**会話調・丁寧め**のトーンが刺さる場（XやTwitterよりやや長文OK）
  - ハッシュタグ1個まで・Linktree経由でnoteに誘導
- **現行のThreads人格**:
  - 名前: 「チャック｜AIで毎日に余白を作る人」
  - 自己紹介: 「月80時間の残業→AIでほぼゼロにした会社員 / 「で、自分の仕事にどう効く？」まで噛み砕いて毎日発信中」
- **判断軸**: Threads側を批評型に寄せすぎると、現行のフォロワーが「トーンが変わった」と離れるリスクがある。一方で X とトーンが完全に分離すると、X→Threadsの相互フォロー誘導の動線で違和感が出る

---

## A. プロフィール文（書き換え3案）

### 案A1：「完全人格統一型」（X連動・批評型へ振り切る）

**名前**:

```
チャック｜AI界隈の煽りにツッコむ人
```

**自己紹介**（150字以内）:

```
月80h残業→AIで定時退社にした会社員
"神プロンプト100選"より"原理1個"派
AI業界の煽りに毎朝冷静にツッコみつつ
「で、自分の仕事にどう効く？」を実務目線で翻訳

・AI初心者でもコピペで即実践
・AIニュースを煽り抜きで読み解く
・プロンプト集PDF無料配布

noteで深掘り👇
```

（138文字）

**狙い**: X側と完全に揃える。フォロー導線で違和感ゼロ、両SNS運用が一本化されて管理コストが下がる。

**リスク**: Threadsの「丁寧め・会話調」ユーザーには**やや尖って見える**。現フォロワーの離脱リスク中。

---

### 案A2：「マイルド批評型」（中道・推奨）⭐

**名前**:

```
チャック｜AIニュースを実務目線で翻訳する人
```

**自己紹介**（150字以内）:

```
月80h残業→AIで定時退社にした会社員
毎朝AIニュースを「で、自分の仕事にどう効く？」まで翻訳
"神プロンプト◯選"系の煽りには、ちょっと距離を置く派

・コピペで即実践のプロンプトテンプレ
・実務に効くAI活用を会話調で解説
・プロンプト集PDF無料配布中

noteで深掘り👇
```

（135文字）

**狙い**: 「煽りに距離を置く派」という言い回しで批評ポジションを**ふんわり明示**しつつ、Threadsらしい会話調・丁寧さは維持。X連動の人格を匂わせるが直球の煽りトーンは避ける。

**リスク**: 中庸すぎて記憶に残りにくい可能性はあるが、Threadsの場の特性に合うのでバランスがいい。

---

### 案A3：「生活密着維持型」（現状の微調整のみ・X分離）

**名前**:

```
チャック｜AIで毎日に余白を作る人
```

（現行維持）

**自己紹介**（150字以内）:

```
月80h残業→AIで定時退社にした会社員
「で、自分の仕事にどう効く？」を毎日発信中

・AI初心者でもコピペで即実践
・AIニュースを実務目線で翻訳
・プロンプト集PDF無料配布

X(@mail.chack)でニュース速報＋ツッコミ
noteで深掘り👇
```

（122文字）

**狙い**: Threadsは現状の「生活密着・余白」路線を維持。Xの方で批評型を担当させる**役割分担**を明示（自己紹介末尾で「X = ニュース速報＋ツッコミ」と書く）。

**リスク**: 人格分裂の懸念は残るが、**役割分担として明示**することで「Threadsはこのトーン、Xはこのトーン」とフォロワーが理解できる。最も安全な選択肢。

## D. 判定基準（5/10 W19振り返り時）

- ✅ プロフィール訪問数 +30% / フォロワー純増 +50% 達成 → A2案を継続
- ⚠️ プロフィール訪問数は伸びたがフォロワー減 → トーンが尖りすぎた可能性、A3案へロールバック
- ❌ 数字に変化なし → A2/A3の効果差がないため、X連動を強化する方向で A1 を再検討（W20）

## E. 推奨案

> **🎯 推奨: 案A2「マイルド批評型」**
>
> 理由:
> 1. X側との人格整合をとりつつ、Threadsの会話調を壊さない
> 2. 「煽りに距離を置く派」の言い回しが**現状フォロワーへのソフトランディング**として機能
> 3. 失敗時のロールバック先（A3）が明確
>
> 万一 A2 で5/10時点に数字が伸びない場合、W20初日（5/11）に A3 へロールバック予定。
