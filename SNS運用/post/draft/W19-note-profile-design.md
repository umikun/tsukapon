## 起点となる事実（2026-05-04 ユーザー実機画面で確認）

### 現状のnoteプロフィール

- **名前**: `チャック | AI×業務効率化の翻訳家`
- **自己紹介（140字以内）**:
  ```
  毎日のAIニュースを「結局どう影響するの？」の視点で翻訳。
  ✅ AI業界初心者でも分かる
  ✅ 難しい話も「で、どうすれば？」まで
  ✅ 気軽サクッとキャッチアップ
  ```
- **ヘッダー画像**: 「効率化で忙しい毎日に余白を作るコツ」「毎日追いかけて『自分の仕事にどう効くか』まで噛み砕いてお〜」（W17時代のままで批評ポジと齟齬）

### 仕様

- **自己紹介文字数制限**: **140文字以内**
- **ヘッダー画像推奨サイズ**: **1920 × 1006 px**（実機編集画面で確認）
- **アスペクト比**: 約 1.91:1（X規格 1500×500=3:1とは**異なる**ため、X用cand9をそのまま流用不可。**note用に再書き出し必要**）
- **アバター（プロフィールアイコン）**: ヘッダー左下に円形配置（X側と同じ位置関係）

## D. ヘッダー画像 改装案

### 仕様

- **推奨サイズ**: 1920 × 1006 px（note公式編集画面で確認）
- **アスペクト比**: 約 1.91:1
- **X用cand9（1500×500・3:1）はそのまま流用不可** → note用に再書き出し必要

### デザイン方針

> **Xヘッダー cand9 と同じデザイン精神（"神プロンプト100選"より、原理1個。＋赤強調）を1920×1006で再構成**

#### 案D1：cand9 直系（X完全同期）⭐最推奨

```
[1行目・中大]   "神プロンプト100選"より、
[2行目・超大赤] 原理1個。
```

- 配色: ダークブルー系背景（ネットワーク模様）+ 白テキスト + 「原理1個」のみ赤アクセント
- メリット: X⇄note の世界観統一が完璧。プロフ訪問者がXとnoteを行き来しても**同じビジュアル体験**を受ける
- 制作時間: Canvaで cand9 のデータを 1920×1006 に書き出し直すだけ（5〜10分）

#### 案D2：縦長余裕活用型

note ヘッダーは**X比で縦が広い**（X 500px → note 1006px・約2倍）。この余白を活かして：

```
[1行目・中大]   "神プロンプト100選"より、
[2行目・超大赤] 原理1個。
[3行目・中]     毎朝AIニュースを"なぜ効くか"まで翻訳する人
```

- メリット: X版で削ったサブコピーをnote版で復活させ、note単独訪問者にも世界観を補強
- デメリット: cand9との完全一致が崩れる

#### 案D3：「3点フィルター」型（cand3直系）

```
AI記事を読むときの3つのフィルター
① 数字の出どころは？
② 書いてる人は実運用してる？
③ 半年後も使える？
```

- メリット: noteの読者層は**じっくり読むタイプ**なので3点リストが刺さりやすい
- デメリット: X側との世界観統一は弱まる

> **第一推奨**: **案D1（cand9 直系）**
> **理由**: X↔noteの世界観統一を最優先。OGPカードでX上に表示される時、ヘッダー画像の一部がプレビューされた時にX側とほぼ同じ印象になる。
