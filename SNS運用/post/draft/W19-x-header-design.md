## 起点となる事実

- 現ヘッダー（[[SNS運用/analytics/screenshots/W19-x-profile-day100-before.png]] 参照）:
  - メインコピー: 「AIに振り回される人生にしたくないあなたに今日も役立つコンテンツ届けます」
  - サブコピー: 「AIニュースを毎日追いかけて『自分の仕事にどう効く？』まで噛み砕いて発信しています」
  - **「ニュース束ねる人」型** = W17時代の世界観
- 改装後プロフ文（A1案）: **「AI界隈の煽りに毎朝ツッコむ人 / 元・月80h残業マン → 定時退社へ」**
- W18実証: 批評型ポジションへの転換で profv率3倍・純フォロー6.4倍 → ヘッダーも同じ世界観に揃えれば **profv→フォロー転換率がさらに伸びる余地**

## 案①「ツッコミ役」直球型 ★A1プロフ文と完全整合

**メインコピー（大・中央上部）**:

```
"神プロンプト100選"より、原理1個。
```

**サブコピー（小・中央下部）**:

```
AI界隈の煽りに毎朝ツッコみつつ
"なぜ効くか"だけ拾って発信する人
```

**配色案**:
- 背景: ダークグレー〜黒（#1a1a1a）
- メインコピー: 白（#ffffff）+ 「原理1個」だけアクセント赤（#e74c3c）
- サブコピー: 薄グレー（#cccccc）

**狙い**: A1案プロフ文と完全に揃える。訪問者が「ヘッダー → プロフ文 → ピン」の3点で同じメッセージを3回受け取る構造になる。
**メリット**: 批評型ポジションが一発で伝わる。既存の批評リプ運用と整合
**デメリット**: 「ニュース系」を期待してた既存フォロワーには硬く見える可能性

## 案②「読者の自分ごと化」型 ★初訪問者へのフック最強

**メインコピー（大・中央上部）**:

```
"AIで月100万"系、9割疑う側です。
```

**サブコピー（小・中央下部）**:

```
1割だけ拾って自分の仕事に使う方法、
毎朝AIニュースから書いてます。
```

**配色案**:
- 背景: 白〜オフホワイト（#fafafa）
- メインコピー: 黒（#1a1a1a）+ 「9割疑う」だけアクセント青（#2c5aa0）
- サブコピー: ダークグレー（#555555）

**狙い**: A3プロフ文（読者の自分ごと化型）と相性が良い。初訪問者の「自分と同じ温度感の人だ」という共感を瞬間に作る。
**メリット**: フック力が3案中最強（4/26リプ大爆発と同じ「批評＋自分ごと化」の構造）
**デメリット**: ややネガ寄りに見える。Day Oneプロモのような前向き訴求と並走しづらい

## 案③「3点コンパクト」型 ★W18勝ち型を視覚化

**メインコピー（大・上部）**:

```
AI記事を読むときの3つのフィルター
```

**3点リスト（中央・大きめ）**:

```
① 数字の出どころは？
② 書いてる人は実運用してる？
③ 半年後も使える？
```

**サブコピー（小・下部）**:

```
詳しくはピン留めから👇
```

**配色案**:
- 背景: 黒（#0e0e0e）
- メインコピー: 白（#ffffff）
- 3点リスト: ターコイズ（#1abc9c）+ 白
- サブコピー: 薄グレー（#aaaaaa）

**狙い**: W18連投④の勝ち型「3点コンパクト＋実運用キーワード」をヘッダーで視覚化。**訪問者にいきなり"批評の方法論"を見せる**ことで「この人から学べそう」感を作る。
**メリット**: 連投シリーズ⑤⑥との世界観統一。保存欲求も誘発しやすい
**デメリット**: 情報量が多めなのでスマホでの可読性要検証。アバター位置との重なりに注意

## 制作・投入手順

### Phase 1: 制作（5/4夜 or 5/6〜5/8 のいずれか）

#### 推奨ツール

- **Canva**（無料・テンプレ豊富）: 「Twitter ヘッダー」テンプレで 1500×500px 即作成可
- **Figma**（無料・自由度高）: フォント・配色を厳密に制御したい場合
- **Photoshop / Affinity**: 既存スキルがあれば

#### フォント推奨

- 日本語: **Noto Sans JP Bold**（無難・可読性高）or **源ノ角ゴシック Heavy**（締まる）
- 英数字: **Inter Bold** or **Helvetica Neue Bold**

#### セーフエリア注意

- **左下から250×250px はアバターで隠れる**前提でデザイン
- 中央上部 1000×200px くらいにメインコピーを集約
- スマホでは中央500×300pxくらいしか見えないことを意識

### Phase 2: 投入（5/9 土曜の夜推奨）

- 5/4（月）プロフ文＋ピン投入 → 5/4〜5/8 の5日間で profv率を計測（ベースライン）
- 5/9（土）夜にヘッダー画像投入 → 5/9〜5/10の2日間で profv率を計測（A/Bテスト）
- 5/10（日）の W19振り返りで「ヘッダー変更で profv率がさらに伸びたか」を判定
- 撮影:
  - 投入前: `W19-x-header-day108-before.png`（5/9 投入直前）
  - 投入後: `W19-x-header-day108-after.png`（5/9 投入直後）

### Phase 3: 効果が出なければW20で再差し替え

5/10 振り返り時点で profv率が伸びていなければ、W20（5/11〜）で別案へ差し替え：
- 案①投入 → 効果薄なら → 案③へ
- 案②（読者の自分ごと化型）は最終手段として温存
