## 🏆 採用案（cand9・最終版）— 2026-05-04 確定

> **採用ファイル**: [[SNS運用/post/draft/W19-x-header-final.png]]
> **由来**: 案①「ツッコミ役」直球型 + 改善（メインコピー1本勝負・サブコピー削除・"原理1個"超大判赤強調・2行段落構造）

### 最終ビジュアル仕様

```
[1行目・中大]   "神プロンプト100選"より、
[2行目・超大赤] 原理1個。
```

- 配色: ダークブルー系背景（ネットワーク模様）+ 白テキスト + 「原理1個」のみ赤アクセント
- サイズ: 1500 × 500 px（X規格）
- 構成: メインコピーのみ、サブコピーなし

### 採用根拠

1. **A1プロフ文と完全呼応**: A1案「"神プロンプト100選"より"原理1個"派」と**ヘッダーが直接引用**する形になり、プロフ訪問者に同じメッセージが3回反復される構造（ヘッダー→プロフ→ピン）
2. **スマホでの可読性最大化**: サブコピー削除により、メインコピーが大判で表示され、5秒で世界観が伝わる
3. **アバター被り問題を完全解消**: cand1ではサブコピーがアバター位置に被っていた問題が、cand9では発生しない
4. **「原理1個」の赤＆超大判**で視線誘導が一発で効く（W18の4/26リプ大爆発のキーフレーズ「原理1個咀嚼」を視覚化）

### スマホプレビュー検証結果

[[SNS運用/analytics/screenshots/W19-x-profile-day100-afterSP_after.jpg]] にて検証済：

## 案①「ツッコミ役」直球型 ★A1プロフ文と完全整合

**メインコピー（大・中央上部）**:

```
"神プロンプト100選"より、原理1個。
```

**サブコピー（小・中央下部）**:

```
AI界隈の煽りに毎朝ツッコみつつ
"なぜ効くか"だけ拾って発信する人
```

**配色案**:
- 背景: ダークグレー〜黒（#1a1a1a）
- メインコピー: 白（#ffffff）+ 「原理1個」だけアクセント赤（#e74c3c）
- サブコピー: 薄グレー（#cccccc）

**狙い**: A1案プロフ文と完全に揃える。訪問者が「ヘッダー → プロフ文 → ピン」の3点で同じメッセージを3回受け取る構造になる。
**メリット**: 批評型ポジションが一発で伝わる。既存の批評リプ運用と整合
**デメリット**: 「ニュース系」を期待してた既存フォロワーには硬く見える可能性

## 制作・投入手順

### Phase 1: 制作（5/4夜 or 5/6〜5/8 のいずれか）

#### 推奨ツール

- **Canva**（無料・テンプレ豊富）: 「Twitter ヘッダー」テンプレで 1500×500px 即作成可
- **Figma**（無料・自由度高）: フォント・配色を厳密に制御したい場合
- **Photoshop / Affinity**: 既存スキルがあれば

#### フォント推奨

- 日本語: **Noto Sans JP Bold**（無難・可読性高）or **源ノ角ゴシック Heavy**（締まる）
- 英数字: **Inter Bold** or **Helvetica Neue Bold**

#### セーフエリア注意

- **左下から250×250px はアバターで隠れる**前提でデザイン
- 中央上部 1000×200px くらいにメインコピーを集約
- スマホでは中央500×300pxくらいしか見えないことを意識

### Phase 2: 投入（5/9 土曜の夜推奨）

- 5/4（月）プロフ文＋ピン投入 → 5/4〜5/8 の5日間で profv率を計測（ベースライン）
- 5/9（土）夜にヘッダー画像投入 → 5/9〜5/10の2日間で profv率を計測（A/Bテスト）
- 5/10（日）の W19振り返りで「ヘッダー変更で profv率がさらに伸びたか」を判定
- 撮影:
  - 投入前: `W19-x-header-day100-before.png`（5/9 投入直前）
  - 投入後: `W19-x-header-day100-after.png`（5/9 投入直後）

### Phase 3: 効果が出なければW20で再差し替え

5/10 振り返り時点で profv率が伸びていなければ、W20（5/11〜）で別案へ差し替え：
- 案①投入 → 効果薄なら → 案③へ
- 案②（読者の自分ごと化型）は最終手段として温存
