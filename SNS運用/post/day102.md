> - **フック文（大）**: ニュースが流れてこない週
> - **サブ文（小）**: 自分が何を持ってるかが浮き彫りになる

```
連休最終日。

1週間"AIニュースに頼れない"環境で、自分の言葉だけで投稿してみた結果。

①最初の2日は「書くことない」と感じた
②3日目から、過去の試行錯誤が言語化されはじめた
③5日目には、ニュース駆動より深い問いが出るようになった

ニュースが流れてこない週は、自分が何を持ってるかが浮き彫りになる。
頼れる情報が無い時、初めて「自分の蓄積」が試される。

連休、嫌いじゃなくなった。
普段ノイズで埋もれてた問いが、ようやく浮き上がる時間。
```

> **メモ:** GW最終日。GW総括として「自分の言葉だけで書く実験」の結果報告。連休の使い方が3段階で変わった話。

## 投稿②(夜)

```
数字より型。

impが伸びなくても、型が固まった日の方が3ヶ月後に効く。

①数字に振られると、毎日が試合
②型に集中すると、毎日が練習
③練習を100回やった人だけが、本番で勝てる

数字を見ない、と言いたいわけじゃない。
数字を見るタイミングを「型ができた後」にずらす、という話。

最初の3週間は、型に集中する。
```

> **メモ:** GW最終夜。数字に振り回されない型重視のメッセージ。「練習」と「試合」の比喩で内省。
