## なぜv2を作ったか

W17の実績で、AIニューススレッドの原ポストはほぼ全滅（imp 4〜15）。
**毎日5本+スレッド**を続けても伸びないことが確定した。

→ **週2回（月・木）に絞り、1本あたりの密度を3倍にする**方向へ転換。

## v2の設計原則

### 旧テンプレの問題

| 旧テンプレ | 何が起きていたか |
|---|---|
| ニュースを5つ並列で紹介 | 1個も刺さらず最後まで読まれない |
| 各ニュースに「実務TIPS」を添える | "TIPS"の表現が一般論で他と差別化できず |
| ハッシュタグ多め | アルゴリズムに「ニュースbot」と認識される |

### v2の3部構成

```
[要約] その週で本当に効く1本だけを最初に提示
　↓
[一般論じゃない見解] 自分の実務文脈での1次解釈を1段
　↓
[読者の不安を1つ植える] 「やらないと来週これに困る」を具体提示
```

5本詰める代わりに、**「1本×深さ3倍」**で勝ちにいく。

## v2 投稿テンプレ（5投稿構成）

### 1/5（フック・要約）

```
今週のAIニュース、本当に効くの1本だけ言うと

【ニュース見出しを20字以内で】

これ、〇〇な人にだけ直撃する話で、自分の現場でもすでに〇〇が起きてます🤔

順に解説します👇
```

**ルール**:
- 1文目に「効くの1本」「実は」「本当は」など希少性訴求語を必ず入れる
- 自分の現場 or 同僚事例を1つ匂わせる
- 絵文字は1つ、語尾は断定しない

### 2/5（一般論じゃない見解）

```
このニュース、ほとんどの解説が「〇〇が便利になった」で止まってるんですけど

実際は **〇〇な業務をしてる人ほど不利になる** 構造があると思ってて

理由は3つ：
①〇〇
②〇〇
③〇〇
```

**ルール**:
- 「ほとんどの解説が〇〇で止まってる」フレーズで批評ポジを匂わせる
- 理由は3つに絞る（4つ以上は読まれない）

### 3/5（読者の不安を1つ植える）

```
で、これ放置すると来月以降、こうなります：

❌ 〇〇が遅い人として目立つ
❌ 〇〇のコスト比較で負ける
❌ 上司から「なんでまだ〇〇してるの？」が来る

特に最後のやつ、地味に効いてくる気がする😭
```

**ルール**:
- 必ず「上司／同僚／会社」の関係性キーワードを最低1つ
- 不安は3つまで、絵文字は1つ

### 4/5（具体的アクション）

```
じゃあ何すればいいか、今週中の最小アクション：

✅ 〇〇を15分だけ試す
✅ 〇〇のドキュメントを1ページだけ読む
✅ 〇〇の比較記事を1本ブクマしておく

全部15分以内。週末に詰め込めば終わります
```

**ルール**:
- アクションは3つ・全部15分以内に収まること
- 「読む」「ブクマ」「試す」のいずれかで構成

### 5/5（誘導＋note）

```
詳細はnoteにまとめました。今日の内容をさらに5本のニュースで補強してます👇

🔗 [当日の記事URL]

このアカウント、AI界隈の煽りに毎朝ツッコみつつ自分の仕事に効く部分だけ抜き出す人なので、フォローしてもらえると次のスレッドも届きます
```

**ルール**:
- 最終投稿に必ずnote URLとプロフ誘導を両方
- ハッシュタグは付けない（v2では削除、bot認識を避ける）

## 月・木の使い分け

### 月曜（4/27, 5/4...）：「先週末の動きを束ねて1本」

- 土日の海外ニュース・OpenAI/Anthropic/Googleの動きから1本
- 主題: 「週明けにこれを知ってると差がつく」

### 木曜（4/30, 5/7...）：「週中の最大トピックを1本」

- 月〜水の中で最も実務影響が大きい1本
- 主題: 「週末までにこれを試しておくと月曜が変わる」

---

## 計測ポイント

- 1/5 と 5/5 の imp 比（脱落率）
- 3/5 の RT率（不安喚起の刺さり度）
- 4/5 の保存率（具体アクションの保存価値）
- 5/5 のリンククリック率（note誘導効率）

→ 週2回のうちパフォーマンス低い方の構成を翌週改善する循環。
