## 投稿①（朝）

> 📸 画像を1枚追加

> 🖼️ **サムネイル文言**
> - **フック文（大）**: やめなかった理由は「楽しい」じゃない
> - **サブ文（小）**: 仕組みに頼ると、嫌な日も手が動く

```
土曜の朝、ふと「3ヶ月もよく続いたな」と思った。

理由を考えてみたら、「楽しいから」じゃなくて「仕組みがあるから」だった。

①前夜にネタの種を1つ置く
②朝7時にアラームで起きる
③10時までに何か出す（質は問わない）

モチベに頼った瞬間、続かない。
仕組みに頼ると、嫌な日も手が動く。

「楽しい」は結果。続けてる人だけが、半年後に「結構楽しいな」と気づく。
```

> **メモ:** W18振り返り着手日。3ヶ月継続の生身を素直に書く。「楽しい」を否定しないけど、最初から楽しいわけじゃないという生身のニュアンス。

## 投稿②（夜）

```
1週間続けて見えた1つ：「投稿の質より、投稿した事実」の方が翌週効く。

質を100にしようとした日、結局出さなかった日が3回あった。
60で出した日は、来週の自分が「何書こう」で迷わない。

「今日は気分が乗らないから明日」を1回許すと、その癖が3週間続く。

60でいい。出した事実が、来週の60を作る。
```

> **メモ:** W18振り返りの最中。投稿の質を上げる試行を全部やってきたが、結局「出した事実の積み上げ」が一番効くという気づき。
