## 投稿①（朝）

> 📸 画像を1枚追加

> 🖼️ **サムネイル文言**
> - **フック文（大）**: 批評型は性格じゃない
> - **サブ文（小）**: 観察した時間の量で決まる

```
W19が終わる土曜。

ルーチンB(批評型リプ)を2週間続けて見えた1つ。

「批評型」は性格じゃなくて、観察した時間の量だった。

①最初の3日は「何書こう」で詰まった
②1週間で同業者の癖が見えた
③2週間で「あ、また同じ煽りだ」と気づけるようになった

"鋭い人"に見える人は、ただ毎日同じ景色を観察してるだけ。
性格じゃなくて、習慣。

性格を変えなくても、観察時間を増やせば自然に変わる。
```

> **メモ:** ルーチンB 2週運用後の振り返り。「性格じゃなくて習慣」の言い切りで、続ければ誰でもできるという背中押し。

## 投稿②（夜）

```
来週の決め方は3行。

①これ続ける（1個）
②これやめる（1個）
③これ試す（1個）

それ以上は手が動かない。

「やること30個」は来週月曜に1個も終わらない。
「3つだけ」は来週金曜には全部終わってる。

決断の質より、決断する数を絞る。
```

> **メモ:** 翌週準備の王道3ステップ。「決断疲れ」回避を、シンプルに3つで終わらせる。
