# X導師.skill 実践事例：実際のアカウント診断

> 以下はX導師.skillを使って、実際のAIクリエイターアカウントを診断したプロセスです。

## ユーザー入力

「最近Xで99本のオリジナルツイートを投稿し、総インプレッション361万でしたが、エンゲージメント率が0.92%しかありません。問題を診断してください。」

## X導師による診断

### Step 1: 現状確認

あなたのデータ：
- 99本のオリジナル投稿 ÷ 46日 = 2.2本/日（推奨の3〜5本を下回っている）
- 総インプレッション361万 → ある程度の基盤があることを示している
- エンゲージメント率0.92% → 健全値の2%を下回っている

### Step 2: ボトルネックの診断

診断フレームワークに沿って各層を確認：

**アルゴリズム層** ✅：インプレッションがあるため配信は正常、アルゴリズムの問題ではない

**コンテンツ層** ⚠️ コアの問題：
- 50%のツイートが「個人の気づき・意見」——エンゲージメント率が最も低いタイプ（0.2〜0.5%）
- プロジェクト型コンテンツ（橙皮書、女娲skill）はエンゲージメント率2〜3%——**気づき型の5〜10倍**
- スレッドがゼロ——3〜5倍のエンゲージメント増幅器を活用できていない

**データ検証**：
| コンテンツタイプ | 代表ツイート | エンゲージメント率 |
|---------|---------|-----|
| Build in Public（橙皮書オープンソース） | 304K views | 2.8% |
| 製品リリース（女娲+乔布斯skill） | 234K views | 1.2% |
| 個人の気づき（自己ポジショニング） | 1,957 views | 0.4% |

**フォロワー層**：
- ブックマーク/いいね比 = 1.15（>1 = 価値型コンテンツ、良いシグナル）
- 読者はコンテンツの価値を認めているが、「エンゲージメントのトリガー」が不足している

### Step 3: アクションプラン

ボトルネックの結論：**コンテンツの質の問題ではなく、コンテンツフォーマットの問題**。

1. **純粋な気づき投稿を削減**：気づきを書く場合は必ず具体的なプロジェクト・数字・成果物と紐づける
2. **週1〜2本のスレッドを投稿**：プロジェクトの経験を8〜10ツイートのスレッドとして構造化する
3. **投稿頻度を上げる**：2.2本/日から3〜4本/日に増やす
4. **エンゲージメントトリガーを追加**：ツイートに質問・フィードバック依頼・議論を呼ぶ意見を盛り込む

期待効果：エンゲージメント率が0.92%から2%以上へ、総インプレッション数が変わらない状態でエンゲージメント数が約2倍になる。
