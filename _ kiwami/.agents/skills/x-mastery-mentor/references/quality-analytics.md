## 品質チェックリスト

作成済みのツイート/Threadをレビューする際に、各項目を確認：

### ツイートチェック
- [ ] Hookは2行以内で注目を引けているか？
- [ ] 「誰に向けて／何を伝えるか／なぜ読む価値があるか」に答えているか？
- [ ] 具体性があるか？（数字、時間、人名）
- [ ] 返信を引き出せるか？（いいねだけでなく）
- [ ] 外部リンクを含んでいないか？（必要な場合は最初の返信に記載）
- [ ] 発信時間はターゲット層の活動時間帯に合っているか？

### Threadチェック
- [ ] 最初のツイートは単体で成立し、かつ引きつける内容か？
- [ ] 1/3/1のリズムに従っているか？
- [ ] 各ツイートはコンテンツを前進させているか？（Rate of Revelation）
- [ ] TL;DRのまとめがあるか？
- [ ] 明確なCTAがあるか？
- [ ] 長さは8〜12ツイートの範囲内か？
- [ ] 大きな文章ブロックでなくbullet pointsを使っているか？

### コンテンツ戦略チェック
- [ ] 今週は少なくとも1本のThreadを投稿したか？
- [ ] 大手アカウントのコメント欄に高品質な返信を残したか？
- [ ] newsletterへ誘導するCTAがあるか？
- [ ] 今週のAIトレンドに反応したか？
- [ ] 単発ツイートとThreadの比率は適切か？

## アンチパターンと落とし穴

### 成長の罠（踏まないこと）

1. **フォロワー購入／相互フォロー・いいねグループ**：短期的にはデータが良く見えるが、長期的にTweepCredが急落。アルゴリズムが不自然なインタラクションパターンを検知して降格させるため、得策ではない
2. **ツールおすすめまとめだけの投稿**：AIジャンルはすでにレッドオーシャン。「10 AI tools you need」的な投稿は誰もが発信しており差別化がない。発信するなら実測データと独自の視点を加えること
3. **海外AIニュースの翻訳のみ**：差別化がゼロ。やるなら自分の意見を加える——「なぜこれが日本の開発者に重要か」や「実際に試してみたら、結果は...」
4. **トレンド追いすぎてポジションを失う**：すべてのトレンドに乗ると、フォロワーがあなたの専門性を見失う。トレンドは選別すること：自分のポジションに関連するものだけに乗る
5. **Hookだけで中身がない**：釣りタイトルは短期的には有効だが、長期的にフォロワーが離れる。Hormoziが言う「超過デリバリー」——1つを約束して3つを提供する
6. **投稿して返信しない**：会話の重みは150x、返信しないのはアルゴリズムの最大レバレッジを放棄すること
7. **Threadが長すぎてまとまらない**：15ツイートを超えると離脱が始まる。Sweet spotは8〜12ツイート

### プラットフォームレベルのリスク（要注意）

- **エンゲージメント率の全体的な低下**：2024〜2025年にXのプラットフォーム全体のエンゲージメント率が48%急落。あなたの問題ではなく、プラットフォームのトレンド
- **Pay-to-playの加速**：非Premiumユーザーのオーガニックリーチは継続的に縮小し、外部リンク投稿はほぼゼロ。Premiumはオプションではなく必須
- **ユーザー移行**：一部のクリエイターがBluesky/Threadsへ分散。ただし現時点ではXがAI/テック系コンテンツの主戦場

## データ振り返りガイド

### 主要指標（優先度順）

| 指標 | 見るポイント | 健全値の参考 |
|------|------------|------------|
| Engagement Rate | インタラクション数/インプレッション数 | >2%良好、>5%優秀 |
| Reply率 | 返信数/インプレッション数 | 高いほど良い（アルゴリズム最重視） |
| Profile Visit率 | プロフィール訪問/インプレッション数 | >1%はあなたへの関心のサイン |
| フォロワー増加数 | 週次の純増フォロワー数 | 冷起動期は日平均5〜10、成長期は日平均20〜50 |
| ブックマーク率 | ブックマーク数/インプレッション数 | ブックマーク高=高価値コンテンツ |
| Newsletterへの誘導 | 新規購読者数/週 | あれば良し、継続してコンバージョン率を追跡 |

### 振り返りのリズム

- **毎日**：前日に投稿したコンテンツのデータを確認し、500インタラクション超の高パフォーマンス投稿をマーク
- **毎週**：今週のトップ3ツイートを分析し、共通点を抽出→テンプレートライブラリを更新
- **毎月**：フォロワー増加曲線・コンテンツタイプ分布・Newsletter成長をレビュー。翌月のコンテンツ戦略を調整

### 診断フレームワーク（ツイートのデータが悪い時）

順番にチェック：
1. **アルゴリズム層**：Premiumは有効か？投稿時間は適切か？外部リンクはないか？
2. **Hook層**：最初の2行に好奇心ギャップはあるか？信頼性のアンカーはあるか？
3. **コンテンツ層**：各ツイートは内容を前進させているか？1/3/1のリズムがあるか？
4. **オーディエンス層**：Engagement Velocityを発動させるのに十分なフォロワー数があるか？まずコメント欄でリーチを借りること

## レポートHTMLテンプレート要件

診断レポートはエコノミスト/新聞スタイルのレイアウトを使用し、以下を必ず含める：
- **ビジュアルスタイル**：セリフフォント（Georgia）、温かみのある紙色背景（#f5f0e8）、赤色アクセント（#C7000A）、グリッドレイアウト
- **データビジュアライゼーション**：ECharts.js（CDN経由）を使用し、少なくともトピック分布図・時間分布図・エンゲージメントファネルを含める
- **必須セクション**：
  1. Banner + Masthead（一文のコアな発見をタイトルとして）
  2. KPI Grid（4つの主要指標を大きな数字で表示）
  3. Lead（サマリー段落、イタリック体、左側に赤いborder）
  4. コンテンツROI分析（トピック別に分類したエンゲージメント比較）
  5. 伝播ファネル（いいね率/ブックマーク率/リツイート率/返信率）
  6. 時間分析（最適な投稿時間帯、投稿リズムの推移）
  7. ブランドナラティブ（ナラティブロール分布とエンゲージメントパフォーマンス）
  8. Top 5アクション提案（番号付き赤丸 + タイトル + 本文 + データ根拠）
  9. Footer（データ期間、サンプル数、分析日時）
