### 重要ルール

- **Engagement Velocity**：最初の15〜30分のエンゲージメント速度でツイートの成否が決まる。15分以内に10以上のエンゲージメント→指数的拡散、3未満→ツイート死亡
- **時間減衰**：6時間ごとに可視性が半減
- **Premium必要性**：フォロワーFeedへの4倍加算 + 非フォロワーFeedへの2倍加算 + TweepCred即時+100。非Premiumでリンクつきツイートのエンゲージメントはほぼゼロ（2026.03データ）
- **Grokトーンスコア**：2025年新規追加、ポジティブ/建設的なコンテンツがより多く配信される
- **外部リンクの解決策**：本ツイートにリンクを置かず、最初のリプライにリンクを置く

### 花叔（Huashu）のポジショニング提案

調査に基づき、花叔（Huashu）のXにおける最適な差別化ポジション：

**「中国のインディペンデント開発者が、AIでプロダクトを作り、その過程を世界に向けて発信する」**

理由：
1. **独自の視点**：中国AIエコシステムの一次情報（DeepSeek、GLMなど）は国際的な読者にとって独自の価値がある
2. **Build in Public との相性が自然**：小猫補光灯のApp Store有料ランキング1位の話は、英語圏のXで大きな拡散ポテンシャルがある
3. **Learn in Public の重ね合わせ**：30万人以上の中国語フォロワーを持つコンテンツ制作の経験を英語の方法論として提供できる
4. **プロダクトによる実証**：ship or shut up——AIテック分野ではプロダクトを作れるかどうかが問われる、花叔（Huashu）にはプロダクトの実績がある

**コンテンツ戦略提案**：
- 60%英語（主戦場）、40%日本語・中国語（既存フォロワーへのサービス）
- 英語コンテンツは中国語の翻訳ではなく、文脈に合わせて新たに書き直す
- 新モデルリリース時は中英同時対応（中国語速評 + 英語深掘りスレッド）

### 中国人開発者の海外展開注意事項

1. 英語ライティングは完璧でなくてよい——AIテック分野は非ネイティブスピーカーに対してより寛容
2. 投稿時間は北米のフォロワーに合わせる：Pacific Time 8〜10 AM（北京時間 23〜01時）
3. オープンソースへの貢献が最良の国際的信頼資産
4. 二言語アカウントは分けて運営し、混用しない
5. 中国AIの一次情報が差別化の武器になる
