# @09pauai「ホームレスから月収1,000万円再現プラン」を冷静に解剖する

> **TL;DR**
> @09pauai が4/27昼に投下した7連投は **「①高単価アフィ → ②Claude Codeで全自動化 → ③10万→100万→1000万のスケール → ④コンテンツ販売で安定化」** という**典型的な高額情報商材ファネルの勧誘構造**。各要素を切り出すと「高単価ジャンルのリストアップ」だけは事実だが、**Claude Codeでの全自動アフィ運用は2026-01のX規約改定でほぼ全面アウト**、**月収1,000万円帯はアフィ全体の0.01〜1.1%（JAO調査）**、**「100万円×10人」モデルは国民生活センターが警告する典型的な高額コンサル誘導の構造そのもの**。**コピーする側のリスクが極端に高い**ので、ネタとして観察する以上の取り扱いは推奨しない。

## 🧭 背景 / なぜ今このスレッドが話題なのか

@09pauai は自称「アフィリエイト10年・最高月収1,000万円」のXアカウント。**2026年4月に入ってから「初心者向け再現プラン」系の連投を立て続けに投下している**（4/24スレッズ×アフィ、4/26「初心者に戻るならこれをやる4つ」、4/27「ホームレスからの再起プラン」）。短期間に同じ"再現性"フックを連射する流れは典型的な**コンサル/教材販売ファネルの集客フェーズ**。

このパターンが盛り上がっている2026年春の背景:

1. **X 2026-01改定でリプ収益化が激変**：従来の「リプ欄でのインプ稼ぎ」がほぼ無価値化。**収益はPremiumユーザーのホームTL表示ベース**に変更され、**自動LLMリプAPIの制限・InfoFi系報酬アプリの禁止**も同時に行われた（[XTEP/グリン社解説](https://note.com/grin_corp/n/n480ee3ce7307)）。**従来型の"自動量産"アフィプレイヤーが収益源を失い、代わりに"教材販売"側へ収益軸を移している**動きが業界全体で起きている。

2. **生成AI×アフィ量産の"見かけの楽さ"急上昇**：Claude Code/MCPでX下書きまで自動化する事例が2026年初頭以降スレッショルド突破（[Uravation解説](https://uravation.com/media/claude-code-x-auto-post-2026/)）。"スキルを売れる人"にとっては絶好の販売タイミング。

3. **アフィ市場全体は転換点**：矢野経済研究所の[2026年市場展望](https://www.yano.co.jp/market_reports/C67123400)は「AI技術浸透とSNS／ショート動画台頭で大きな転換点」と明記。AI Overviews でSEOトラフィック構造が崩れ、SNSアフィに比重が移っている。

つまり @09pauai のような"古参が再現プランを売る"ムーブは、業界の地殻変動を背景に**過去の成功体験を商材化する最後のチャンス**として量産されているフェーズ。

## 🌐 最新動向（2026-04-27時点）

- **X 2026年1月収益化大改革**：従来のリプ欄インプ稼ぎが無価値化。Premium表示ベースに移行し、リワードしきい値が$10→$30に引き上げ。インプレゾンビ対策として**自動LLMリプAPI制限／InfoFi報酬アプリ禁止**が同時施行 — [XTEP/グリン社](https://note.com/grin_corp/n/n480ee3ce7307), 2026-01
- **「Premiumホームタイムライン表示」が収益単位に**：従来の総インプではなく、**Premium会員ホームに到達した数のみカウント**。古参インフルエンサーの収益が一夜で半減した報告多数 — [チャエン解説](https://note.com/chaen_channel/n/nfda81a600a1b), 2026-02
- **アフィ市場：月3万円以上のプレイヤーは9.4%（10年で4倍）**ながら**月100万以上は1.1%のまま**。下位の裾野は広がったが上位は変わらず — [JAO 2023調査](https://www.japan-affiliate.org/news/survey2023/), 2023-05公表（最新の更新値も同水準が継続）
- **生成AI×SEOで「アフィ記事の検索流入」が急減**：AI Overviewsが上位答えを直接表示、長文SEO型アフィの収益モデル崩壊が進行 — [矢野経済研究所](https://www.yano.co.jp/market_reports/C67123400), 2026-Q1
- **国民生活センター**：情報商材相談は2022年7,000件→2024年4,118件と減少傾向。だが**SNS発の高額コンサル誘導手口は引き続き相談対象の主要部分**。「ホームページで容易に稼げる事例を見せる→クリック誘導→ミニ講座→高額契約」のファネルが警告対象 — [国民生活センター](https://www.kokusen.go.jp/soudan_topics/data/infoproducts.html), 2024年通期データ
- **Claude Code×SNS自動化のbest practice**：完全自動投稿ではなく **「AIが下書き→人間が確認→投稿」**が公式ベンダーの推奨ライン。Apiyi社が2026-03のフィールドテストでアカウントBAN事例を多数確認 — [Apiyi BAN対策ガイド](https://help.apiyi.com/en/claude-account-ban-prevention-china-2026-guide-en.html), 2026-03

## 🧩 関連概念・隣接分野

- **高額情報商材ファネル**: 「フリーで価値提供 → 中価格教材 → 高額コンサル → サブスク化」の階段。 @09pauai の7連投はまさに **第1段の"無料の魅せフェーズ"**。経済評論家・消費者団体が批判する典型構造（[東洋経済](https://toyokeizai.net/articles/-/421978)）
- **インプレゾンビ問題**: 2024〜2025年に多発した、AIで意味のないリプを大量生成して収益化する手法。**2026-01のX改定で経済価値が消滅**。@09pauai の「Claude Codeで全自動化」プランはこの規制の真正面を行く設計
- **自動アフィ運用とX規約の摩擦**: 「複数アカウントで同一/類似投稿」「トレンドの自動操作」「無関係宣伝」が明文で禁止。違反時はAPI停止＋凍結 → [[_ memory/short-term.md]] にも「@09pauai 流の戦略は古い」とメモ済み
- **"ストーリーテリング型コンテンツ販売"の倫理線**: 「ホームレスから〜」のような物語型フックは**実体験を含むなら正当、創作なら景表法・特商法の優良誤認**に該当しうる
- **Threads × アフィの直リンク戦略**: 同じ作者が4/24に推奨していたモデル（[[Clippings/Post by @09pauai on X 1.md]]）。Threadsは2025年〜2026年にかけてアフィ直リンクを許容している過渡期にあり、規約が変わる前の"先行者利得"狙いに近い

## 🪞 反対意見 / 別角度からの視点

- **肯定側の主張（@09pauai 寄りの読み方）**:
  - 高単価ジャンル選定は**事実として正しい**（CPA $50〜$500の市場は存在）
  - AIによる下書き量産で**作業時間が10分の1**になるのは事実
  - "PDCAを高速で回す"原則は**マーケティングの基本**として正論
  - 「実績を作って教材化」自体は**合法的なビジネスモデル**の1パターン

- **否定 / 慎重派の主張**:
  - **再現性の欺瞞**: 「1ヶ月で月10万・1年で月1,000万」は **0.01〜1.1%の上澄みの結果論**を「再現可能」とミスリード
  - **規約リスク**: Claude Codeでの全自動アフィ運用は**X 2026-01改定とX自動化ルールの両方に抵触**。1ヶ月持たずに凍結する確率が高い
  - **誘導構造の危険**: 「100万円で10人にコンサル」は国民生活センターが繰り返し警告する**SNS高額契約トラブル**の典型構造
  - **古い武勇伝の再現可能性**: 過去にアフィで成功した手法は**SEO/SNSアルゴリズム/規約の変更で陳腐化**するスパンが短い。10年前のノウハウが2026年に効く保証はない
  - **「自分がそうだった」自白問題**: 本人が「コンサルで稼いでも一発屋になりがち（実際に僕がそう）」と6/7で告白している。**読者の前提として"このプラン自体が一発屋構造"であることを当人が認めている**

- **中立的に見るときの補助線**:
  - **観察対象としては優秀**: 4/26〜4/27の連投は「**現代版・高額商材ファネルの教科書**」として批評コンテンツの素材になる
  - **数値だけは追う価値あり**: 表示数の推移（144→47→43→23→27→64）は**フック設計の研究材料**として有用
  - **"自分はやらない"線で批評型ポストの素材化** = vault運用としては最適。[[SNS運用/analytics/W18戦略メモ.md]] の **B項「批評型リプを日3本」** や **C項「連投シリーズ」** の格好の素材
  - **同種戦略の研究は [[調査/2026-04-25-x-reply-impression-strategy.md]] / [[調査/2026-04-25-threads-affiliate-side-hustle.md]] とセットで読むと立体的**
