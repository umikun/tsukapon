このせいで Notion AI を削除します。

これを claude-obsidian と呼び、Obsidian を実行中のノートテイカーに変えてくれます。あなたが作業している間、ファイルの整理、相互参照の管理、そして自己維持を行います。

他のすべての Obsidian AI プラグインとの違いは、アーキテクチャにあります。Smart Connections はあなたのノートの上にチャットインターフェースを置くだけです。これ（claude-obsidian）は、ノートそのものを生成、整理、進化させる知識エンジンです。

→ あなたが取り込むすべてをエンティティ、コンセプト、ソースに自動整理

→ あなたのボールト全体で矛盾をフラグ付けし、ソースにリンクするコールアウトで指摘

→ 自律的な3ラウンドのウェブリサーチを実行し、出力を自動的にファイル化

→ 複数のソースに対する並列バッチ取り込み

→ クエリが特定のウィキページを引用し、曖昧な類似性マッチではない

→ ネイティブの Obsidian Bases ダッシュボードに加え、ビジュアルキャンバスのコンパニオン

→ クロスプロジェクトのセットアップにより、1つのボールトがすべての Claude Code プロジェクトにフィード
