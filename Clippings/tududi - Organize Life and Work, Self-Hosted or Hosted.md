## Self-Host tududi

Want full infrastructure control? Run tududi on your own server. It's free, open-source, and Docker-ready.

### Docker Setup

Deploy in minutes with Docker

1

#### Pull the Docker image

```
docker pull chrisvel/tududi:latest
```

2

#### Run the container

```
docker run \
  -e TUDUDI_USER_EMAIL=you@example.com \
  -e TUDUDI_USER_PASSWORD=yourpassword \
  -e TUDUDI_SESSION_SECRET=$(openssl rand -hex 64) \
  -v ~/tududi_db:/app/db \
  -p 3002:3002 \
  -d chrisvel/tududi:latest
```

3

#### Access at http://localhost:3002
